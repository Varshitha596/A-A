# 📌 Project Overview

E-Cart is a responsive e-commerce web application developed using React.js. This project was built with the guidance and learning support from NxtWave. The application allows users to browse products, view detailed information, login authentication, and explore an online shopping interface similar to real-world e-commerce platforms.

The project focuses on:

* React component architecture
* Authentication handling
* API integration
* Routing
* State management
* Responsive UI design

---

# 🚀 Features

## 🔐 Authentication

* Login functionality
* Protected routes
* JWT token handling using cookies
* Error handling for invalid credentials

## 🛍️ Product Section

* Fetching products from APIs
* Displaying product cards
* Product details page
* Prime deals section
* Similar products section

## 🔎 Filters & Search

* Category filters
* Rating filters
* Search functionality
* Clear filters option

## 📱 Responsive Design

* Mobile-friendly layout
* Sidebar navigation
* Adaptive UI for different screen sizes

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone <your-github-repo-link>
```

## 2️⃣ Navigate to Project Folder

```bash
cd e-cart
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```
